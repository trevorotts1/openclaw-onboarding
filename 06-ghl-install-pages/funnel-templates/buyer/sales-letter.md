## When to use / trigger criteria

Long-form **written** sales copy on a single page using the **Star/Story/Solution** script, followed by an order form with a bump and an OTO/downsell sequence. The written equivalent of the VSL Funnel — used when switching sales modalities or when an audience prefers to read rather than watch video.

**Goals:** sell with long-form written copy · serve a read-not-watch audience · run an A/B alternative to a VSL on the same offer · convert high-intent traffic that wants detail, proof, and to skim at its own pace.

**Keywords:** sales letter · long form sales page · written sales copy · text sales page · long copy · Star Story Solution · salespage.

**Signals:** audience prefers reading · offer needs heavy proof/detail/FAQ · client wants a text alternative to test against a VSL · B2B/professional/older audience that skims long copy.

**Route elsewhere when:** audience converts better on video → VSL Funnel · free/just-shipping front end → 2-Step Tripwire · recurring core → Membership/Continuity · physical book → Book Funnel.

## Page structure (3 pages)

1. **Long-Form Sales Letter Page** — Carry the entire sale in written copy via Star/Story/Solution, with anchor nav, proof, guarantee, and an order form with bump at the close.
   - Magnetic headline + deck; **STAR** (introduce the Attractive Character); **STORY** (Epiphany Bridge / problem-to-discovery); **SOLUTION** (offer + unique mechanism + value stack); anchor/jump nav for skimmers; proof + testimonials + case studies; risk-reversal **guarantee**; FAQ/objection section; **order form with order bump** at the bottom CTA; P.S. close.
2. **OTO / Upsell Page** — One-click upsell right after the sale.
   - Interrupt headline; short upsell copy with stacked value; one-click accept + decline; optional downsell.
3. **Confirmation / Thank-You Page** — Confirm, deliver, bond, bridge.
   - Order summary + access; Attractive-Character welcome; next-step CTA into the value ladder.

## Copy framework

- **Primary persona — `edwards-copywriting-secrets`** (Copywriting Secrets): lead copywriter for the long-form page — headline formulas, benefit stacking, specificity rules, objection preemption, P.S. close. The written-copy specialist.
- **Sales script — Star/Story/Solution** (the canonical script this funnel is built on): introduce the **Star** (Attractive Character) → tell the **Story** (Epiphany Bridge) → present the **Solution** (offer + value stack).
- **Supporting — `brunson-marketing-secrets-blackbook`** (Marketing Secrets Blackbook): Attractive Character + Epiphany Bridge — supplies the Star, the Story arc, and the bonding voice.
- **Supporting — `miller-building-storybrand`** (Building a StoryBrand): SB7 clarity filter (customer = Hero, brand = Guide) — keeps the letter customer-centric, not product-bragging.
- **Supporting — `russell-brunson-the-funnel-hackers-cookbook`** (The Funnel Hacker's Cookbook): Sales Letter recipe — anchor nav, proof, guarantee, order-form + bump + OTO placement.

This is the written twin of the VSL Funnel.
