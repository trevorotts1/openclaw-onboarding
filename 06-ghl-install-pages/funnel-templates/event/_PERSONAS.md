# Copy Personas — Event Funnel Group

These are the book-derived copywriter personas referenced by every template's `copyFramework`.
Each persona maps to a specific Brunson/ClickFunnels-canon script or formula set.

| Persona slug | Source book / author | Core script / formula set | Best used for |
|---|---|---|---|
| `perfect-webinar` | Expert Secrets — Russell Brunson | The Perfect Webinar: **Intro/Hook (The One Thing) → 3 Secrets (break the 3 false-belief patterns) → The Stack → The Close (trial + scarcity)** | The live/auto presentation body, any "teach-then-pitch" event |
| `epiphany-bridge` | Expert Secrets / Network Marketing Secrets — Russell Brunson | The Epiphany Bridge story arc: **Backstory → Desires → Wall → Epiphany → Plan → Conflict → Achievement → Transformation** | Origin-story videos, pre-launch content, recruiting presentations |
| `star-story-solution` | DotCom Secrets — Russell Brunson | **Star / Story / Solution** sales script; Attractive Character; Value Ladder framing; Who/What/Why/How | VSLs, sales letters, upsell/OTO order pages |
| `storybrand-sb7` | Building a StoryBrand — Donald Miller | **SB7: Character → Problem → Guide → Plan → Call to Action → Avoid Failure → Success** | Brand-clarity headlines, summit/event positioning, "the customer is the hero" framing |
| `copywriting-secrets` | Copywriting Secrets — Jim Edwards | Big-Promise headline formulas, fascination bullets, hooks, risk-reversal/guarantee blocks, P.S. stacks, button/CTA copy | Registration headlines, opt-in bullets, order buttons, urgency + guarantee blocks |

## Standard closing frameworks (referenced inline)
- **The Stack** (Perfect Webinar) — restate the full value of every component, anchor a high "real value," reveal the actual price.
- **3 Closes** (Follow-Up Funnels) — Emotion close → Logic close → Fear/scarcity close, used in the email/SMS follow-up.
- **Viral Loop** (Network Marketing Secrets) — Epiphany Bridge → Demo → Call to Action + invite-a-friend mechanics, for recruiting/duplication funnels.
