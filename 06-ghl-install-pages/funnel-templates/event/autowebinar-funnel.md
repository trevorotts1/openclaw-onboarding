# AutoWebinar Funnel (Evergreen Webinar)

**id:** `autowebinar-funnel` · **category:** Event Funnels · **library ref:** #16 · **length:** long-form
**aliases:** Automated Webinar Funnel, Evergreen Webinar Funnel, Just-In-Time Webinar Funnel
**source books:** Cookbook, Marketing Secrets Blackbook, DotCom Secrets, Expert Secrets

## When to use / trigger criteria
Use to scale a webinar that has **already converted live** — run the recording evergreen 24/7 with just-in-time sessions, watch-time segmentation, and an automated per-lead cart-close.

- **Goals:** sell continuously without presenting live; scale paid traffic to a hands-off event; segment follow-up by watch behavior.
- **Keywords:** "auto webinar", "evergreen webinar", "on-demand training", "watch now", "just in time", "automated webinar", "replay available".
- **Signals:** a live Perfect Webinar already converts; client wants always-on sales; steady traffic volume; wants per-lead deadlines.
- **Anti-signals:** no proven live webinar yet (run live first); offer/price changes often (recording goes stale); one-time fixed-date event (use Live Webinar).

## Page structure
1. **Registration Page (just-in-time)** — Big-Promise headline; session-time picker incl. "starts in 15 min"; 3 fascination bullets; opt-in form. *Widgets: opt-in form, just-in-time session selector, countdown-to-session.*
2. **Confirmation Page** — countdown to chosen session; add-to-calendar + reminders; short pre-frame video; "show up early." *Widgets: countdown-to-session, add-to-calendar, reminder opt-in.*
3. **Broadcast / Replay Page** — embedded recording as simulated-live; watch-time tracking; buy button revealed at the Stack timestamp; per-registrant cart-open countdown. *Widgets: video embed w/ watch-time tracking, timestamp-triggered order button, per-lead countdown.*
4. **Post-Webinar Offer Page** — Stack recap + bonuses + guarantee; order form with bump; evergreen per-lead deadline (e.g. 48h); FAQ + testimonials. *Widgets: order form + bump, evergreen per-lead countdown, FAQ accordion.*
5. **Segmented Follow-Up (sequences, not a page)** — segments: no-show / left-early / watched-end / watched-no-buy; 3 Closes (emotion → logic → fear) tied to the per-lead deadline; expiring replay re-offer.

## Copy framework
- **Primary persona:** `perfect-webinar` — reuse the proven Perfect Webinar recording (Intro → 3 Secrets → Stack → Close); do NOT rewrite a converting recording.
- **Supporting:** `copywriting-secrets` (registration + per-lead urgency), 3 Closes for segmented follow-up, `star-story-solution` (order-page recap), `epiphany-bridge` (pre-frame).
- The new copy work is the per-lead deadline urgency and the watch-segment follow-up sequences, not the presentation.
