# Live Demo / Home Party Funnel

**id:** `live-demo-home-party-funnel` · **category:** Event Funnels · **library ref:** #23 · **length:** short-to-medium-form
**aliases:** QVC Funnel, Home Party Funnel, FunnelFridays-style Funnel, Live Build Demo Funnel
**source books:** Cookbook, Marketing Secrets Blackbook, Network Marketing Secrets

## When to use / trigger criteria
Use to sell a product by **demonstrating it live** (or building something in real time) with buy buttons under the player, then keep the replay selling. Add the Home Party variant for network-marketing duplication and invite-a-friend virality.

- **Goals:** sell via live product demo (QVC-style) with instant buy buttons; keep the replay selling with persistent links; (NM) drive invite-a-friend duplication.
- **Keywords:** "live demo", "home party", "QVC", "live build", "watch me build", "buy now below", "going live", "invite friends", "party".
- **Signals:** physical/visual product that benefits from live demo; client comfortable going live; NM duplication goal; wants impulse buys during/after a live event.
- **Anti-signals:** complex high-ticket needing a full Perfect Webinar (use Webinar); no live-demoable product (use VSL/Sales Letter); pure list-building (use Squeeze/Lead Magnet).

## Page structure
1. **Party Invite / Registration Page** *(NM variant; plain demos can route straight to broadcast)* — fun social headline + host + "what we'll show/build"; opt-in form; date/time + "bring a friend." *Widgets: opt-in form, countdown to go-live.*
2. **Social-Share Confirmation Page** — "you're in" + add-to-calendar; invite-a-friend share buttons w/ pre-filled message + personal link; incentive to invite; reminder opt-in. *Widgets: add-to-calendar, social-share/referral block, reminder opt-in.*
3. **Live Stream / Broadcast Page** — live embed (FB/IG/YT/Zoom); buy button(s) directly below the player; live chat + invite prompts; limited-time live-only bonus. *Widgets: live video embed, order button/form block, live chat, share prompt.*
4. **Replay Archive Page (buy links active)** — replay embed; persistent buy button(s) below player; highlights/timestamps; testimonials + guarantee. *Widgets: replay embed, persistent order button/form, testimonial block.*

## Copy framework
- **Primary persona:** `epiphany-bridge` — live demo narrative = Epiphany Bridge + Viral Loop (story → demo → call to action), especially the NM Home Party variant.
- **Supporting:** `copywriting-secrets` (buy-button copy, share prompts, live-only bonus urgency), `star-story-solution` (compressed product framing), `perfect-webinar` (one-secret mini-pitch if the demo teaches).
- Energy and live social proof drive this funnel. The NM Home Party variant leans hardest on the Viral Loop; a plain QVC-style demo can start at the broadcast page.
