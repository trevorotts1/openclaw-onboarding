# Perfect Webinar Shortcut (Facebook/Instagram Live Funnel)

**id:** `perfect-webinar-shortcut-funnel` · **category:** Event Funnels · **library ref:** #33 · **length:** short-form
**aliases:** Perfect Webinar Shortcut, FB Live Funnel, IG Live Funnel, No-Slides Whiteboard Webinar
**source books:** Expert Secrets · **proof:** Kaelin Poulin generated $650K from a single session

## When to use / trigger criteria
Use to run a fast, slide-less Perfect Webinar **live on Facebook/Instagram** (whiteboard + origin story + ONE secret + close), then retarget viewers and route warm traffic straight to an order page.

- **Goals:** sell fast without a slide deck or webinar tech; use a live FB/IG session as both content and paid-ad creative; retarget viewers and convert warm traffic.
- **Keywords:** "perfect webinar shortcut", "fb live", "ig live", "facebook live", "whiteboard", "one secret", "quick webinar", "go live and sell".
- **Signals:** wants speed (~15 min build) over polish; comfortable going live with a whiteboard; has ad budget to boost/retarget; single clear offer with one core belief to break.
- **Anti-signals:** offer needs 3 belief breaks + slides (use Webinar); audience won't watch social live (use AutoWebinar/VSL); no retargeting/ads capability.

## Page structure
1. **Live Event Setup / Landing Page** — big-promise headline + go-live time + platform; RSVP/reminder opt-in (optional); "what you'll learn" (one-secret tease); link to the live. *Widgets: RSVP/reminder opt-in, countdown to go-live.*
2. **Live Broadcast (on FB/IG, native)** — whiteboard origin story (Epiphany Bridge); ONE secret (the single biggest belief break); the close + clear "go here to get it" CTA; live comments engagement. *Native platform; link-out CTA tracked via UTM/redirect.*
3. **Retargeting Capture / Bridge Page** — "watched the live? get the offer/replay here"; short bridge recap; pixel/retargeting fires; button to order. *Widgets: bridge opt-in (optional), pixel/tracking embed, CTA button.*
4. **Order Page** — Stack recap + offer + price; order form with bump; urgency (session-only bonus/countdown); guarantee + FAQ. *Widgets: order form + bump, countdown, FAQ accordion.*

## Copy framework
- **Primary persona:** `perfect-webinar` — the live is a Perfect Webinar **Shortcut**: origin story + ONE secret + close, no slides, whiteboard only.
- **Supporting:** `epiphany-bridge` (origin story), `copywriting-secrets` (setup headline, ad copy, bridge recap, order urgency), `star-story-solution` + Stack (order page).
- Only ONE secret (not three) — pick the single biggest false belief. Speed and authenticity beat polish; the raw live IS the ad creative.

## GHL build notes (Skill 6 + Skill 44)
- The live runs on Facebook/Instagram (not a GHL page). **Skill 6** builds the lightweight setup/bridge/order pages and the tracking pixel.
- **Skill 44 widgets:** RSVP/reminder opt-in; countdown to go-live; pixel/retargeting embed; order form with bump; order-page countdown + FAQ accordion.
- **Automations:** RSVP → reminder + "we're live"; viewer pixel → retargeting audience → bridge/order page; purchase → fulfillment, non-buyer → retargeting re-offer.
- **Products:** single offer (often mid-ticket), order bump. Most of the engine is the retargeting ad flow.
