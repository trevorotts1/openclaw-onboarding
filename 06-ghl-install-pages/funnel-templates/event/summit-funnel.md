# Summit Funnel

**id:** `summit-funnel` · **category:** Event Funnels · **library ref:** #22 · **length:** long-form
**aliases:** Virtual Summit Funnel, Online Summit Funnel, Speaker Summit Funnel
**source books:** Cookbook, Expert Secrets, Traffic Secrets

## When to use / trigger criteria
Use to build a large list and authority fast by hosting a **free multi-speaker online event**, monetized by a paid all-access recordings bundle (plus speaker cross-promotion traffic).

- **Goals:** rapid list growth via speaker cross-promotion; authority by association; monetize free registrants with a paid all-access upsell.
- **Keywords:** "summit", "virtual summit", "online summit", "speaker lineup", "free registration", "all-access pass", "sessions", "interviews".
- **Signals:** wants rapid list growth + authority; can recruit multiple promoting speakers; topic supports 10–30+ sessions; wants free front end + paid upsell.
- **Anti-signals:** single-presenter offer (use Webinar); no speaker network; needs an immediate high-ticket sale not list-building.

## Page structure
1. **Summit Registration / Sales Page** — big-promise headline + dates; speaker lineup showcase; free opt-in; agenda + "who this is for." *Widgets: free opt-in form, speaker grid, countdown to start.*
2. **Confirmation + All-Access Upsell (OTO)** — confirmed + add-to-calendar; All-Access Pass pitch (lifetime recordings/bonuses/transcripts); order form with bump; price-rise urgency. *Widgets: add-to-calendar, all-access order form + bump, price-rise timer.*
3. **Daily Session / Live Stream Pages** — per-speaker/per-day session embeds; schedule + "available until midnight" expiry; all-access reminder by the player; speaker bios + lead-magnet links. *Widgets: session embeds, schedule block, all-access CTA.*
4. **All-Access Recordings Member Area** — gated recordings + transcripts + bonuses; speaker resources; backend value-ladder CTA. *Widgets: membership/gated content, late-upgrade order form.*

## Copy framework
- **Primary persona:** `storybrand-sb7` — attendee = hero, host = guide, the summit = the plan to win; keeps the free-event promise simple.
- **Supporting:** `copywriting-secrets` (headline + speaker fascinations + opt-in bullets), `star-story-solution` + Stack (all-access upsell), `epiphany-bridge` (host welcome / session intros).
- Speakers each promote, so the registration page must convert their cold-ish traffic fast.
