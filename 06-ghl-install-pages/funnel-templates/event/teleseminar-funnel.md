# Teleseminar Funnel

**id:** `teleseminar-funnel` · **category:** Event Funnels · **library ref:** #28 · **length:** medium-form
**aliases:** Tele-Seminar, Audio Seminar Funnel, Conference Call Selling Funnel
**source books:** Marketing Secrets Blackbook

## When to use / trigger criteria
Use for a **single-presenter audio/dial-in selling seminar** — a Perfect-Webinar-style teach-then-pitch delivered by voice only (no slides), ideal for audio-native audiences or low-production setups.

- **Goals:** sell via a single-presenter audio teaching call; reach an audio-first audience; run a webinar-style close without slides/video.
- **Keywords:** "teleseminar", "tele-seminar", "conference call", "audio training", "dial in", "live call", "register for the call", "replay line".
- **Signals:** single host, no slides, audio-only; audio-consuming audience; coaching/consulting/network-marketing/faith niches; wants the Perfect Webinar close in audio.
- **Anti-signals:** needs visual demo/slides (use Webinar); multiple speakers (use Telesummit); sub-$100 impulse (use VSL/5-Minute Perfect Webinar).

## Page structure
1. **Registration Page** — curiosity Big-Promise headline + call date/time; 3 fascination bullets (taught by voice); opt-in form (email + phone); host credibility. *Widgets: opt-in form, countdown to call start.*
2. **Confirmation Page (dial-in details)** — dial-in number/link + PIN; add-to-calendar + SMS reminders; pre-frame audio/video + "what you'll learn"; "call 5 min early." *Widgets: add-to-calendar, SMS/email reminder.*
3. **Replay Audio Page** — audio replay player; replay-availability countdown; offer CTA below the player; Stack recap text. *Widgets: audio player, replay-expiry countdown, order CTA.*
4. **Post-Seminar Offer Page** — Stack recap + bonuses + guarantee; order form with bump; cart-close countdown; FAQ + testimonials. *Widgets: order form + bump, cart-close countdown, FAQ accordion.*

## Copy framework
- **Primary persona:** `perfect-webinar` — the call is a Perfect Webinar delivered by voice (Intro/Hook → 3 Secrets → Stack → Close), no slides.
- **Supporting:** `epiphany-bridge` (call open / origin story), `copywriting-secrets` (registration headline/bullets + replay urgency), `star-story-solution` (offer-page recap).
- It is the Perfect Webinar without visuals — verbal storytelling and the verbal Stack carry the whole close. Strong Epiphany Bridge delivery matters even more without slides.
