# Telesummit Funnel

**id:** `telesummit-funnel` · **category:** Event Funnels · **library ref:** #27 · **length:** medium-form
**aliases:** Tele-Summit, Audio Summit Funnel, Interview Series Summit
**source books:** Marketing Secrets Blackbook

## When to use / trigger criteria
Use for a **multi-speaker interview-series event delivered primarily by audio/dial-in**, with a paid all-access recordings upgrade — when video production is impractical or the audience prefers listen-on-the-go.

- **Goals:** run a multi-speaker event without video overhead; reach a listen-while-busy audience; build a list + monetize via paid recordings.
- **Keywords:** "telesummit", "tele-summit", "audio summit", "interview series", "dial in", "listen in", "speaker schedule", "all-access recordings".
- **Signals:** multiple speakers but low video capacity; audio-consuming audience; interview/conversation format; wants list growth + recordings upsell.
- **Anti-signals:** needs visual demo/slides (use Summit/Webinar); single presenter (use Teleseminar); hard high-ticket close required (use Webinar/Application).

## Page structure
1. **Telesummit Registration Page** — big-promise headline + dates; speaker schedule; free opt-in (email + phone); "dial in or stream audio" how-it-works. *Widgets: free opt-in form, speaker schedule, countdown to first session.*
2. **Access Delivery / Confirmation Page** — dial-in details + session links + schedule; add-to-calendar + SMS reminders; all-access upgrade pitch (OTO) with order form + bump. *Widgets: add-to-calendar, SMS reminder, all-access order form + bump.*
3. **Session / Interview Pages** — per-session audio player (or dial-in); speaker bio + free-gift link; availability window note; all-access CTA beside player. *Widgets: audio player, speaker bio, all-access CTA.*
4. **Paid All-Access / Recordings Upgrade Page** — all-access value stack; order form + bump; price-rise/closing countdown; guarantee + FAQ. *Widgets: order form + bump, price-rise timer, FAQ accordion.*

## Copy framework
- **Primary persona:** `storybrand-sb7` — clear promise for a multi-speaker event.
- **Supporting:** `copywriting-secrets` (headline + speaker fascinations), `epiphany-bridge` (host/interview framing), `star-story-solution` + Stack (all-access upgrade).
- Same authority/list mechanics as the video Summit, but copy must sell the audio/dial-in convenience ("listen on your commute"). The recordings upsell carries monetization.
