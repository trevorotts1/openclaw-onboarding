# Webinar Funnel (Live / Perfect Webinar)

**id:** `webinar-funnel` · **category:** Event Funnels · **library ref:** #15 · **length:** long-form
**aliases:** Perfect Webinar Funnel, Live Webinar Funnel, Sales Webinar Funnel
**source books:** Cookbook, Marketing Secrets Blackbook, Network Marketing Secrets, DotCom Secrets, Expert Secrets, Traffic Secrets

## When to use / trigger criteria
Use to sell a mid-to-high-ticket offer ($297–$3,000+) to a warm/semi-warm audience via a **live teaching presentation** that closes with The Stack and a cart-close deadline.

- **Goals:** sell a course/program/service through teach-then-pitch; break the 3 core false beliefs; use live scarcity (cart closes when the replay window ends).
- **Keywords:** "webinar", "perfect webinar", "masterclass", "live training", "register for the webinar", "3 secrets", "the stack", "cart closes".
- **Signals:** offer needs explaining before purchase; client willing to present live; price too high for a simple order page; audience needs belief-shifting.
- **Anti-signals:** client won't present (use AutoWebinar after a winning live one exists); sub-$100 impulse buy (use 5-Minute Perfect Webinar/VSL); multi-speaker (use Summit).

## Page structure
1. **Registration Page** — curiosity Big-Promise headline; date/time selector; 3 fascination bullets teasing the 3 secrets; host credibility; opt-in form. *Widgets: opt-in form, live-time selector, countdown-to-start.*
2. **Confirmation / Indoctrination Page** — add-to-calendar + reminder opt-in; indoctrination video (Epiphany Bridge origin); why-attend-live bonus tease. *Widgets: add-to-calendar, SMS/email reminder, video embed.*
3. **Live Event Page** — live webinar embed; Perfect Webinar flow (Intro → 3 Secrets → The Stack → Close); live chat; buy button revealed during The Stack with cart-open countdown. *Widgets: webinar embed, live chat, reveal-on-cue order button, live countdown.*
4. **Replay / Urgency Page** — replay embed; cart-close countdown (expires = offer gone); stack recap + bonuses + guarantee; order button + FAQ. *Widgets: replay embed, fixed countdown, order form with bump, FAQ accordion.*

## Copy framework
- **Primary persona:** `perfect-webinar` (Expert Secrets) — Intro/Hook "The One Thing" → Secret 1 (vehicle / false belief) → Secret 2 (internal) → Secret 3 (external) → The Stack → Close (trial + scarcity).
- **Supporting:** `epiphany-bridge` (indoctrination + webinar open), `copywriting-secrets` (registration headline/bullets + replay urgency), `star-story-solution` (replay/order stack recap).
- The 3 Secrets each break one false-belief pattern. The Stack is the close. Live scarcity + replay deadline converts fence-sitters.
