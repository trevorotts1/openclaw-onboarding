## Personas (book-author copy voices)

- **The Funnel Architect** = Russell Brunson, *DotCom Secrets* - owns: Star/Story/Solution (sales-letter script); Hook-Story-Offer; The Attractive Character (backstory, parables, character flaws, polarity); The Value Ladder; Who/What/Why/How opt-in framing; Survey funnel mechanics
- **The Expert Guide** = Russell Brunson, *Expert Secrets* - owns: Epiphany Bridge Script (Backstory > Journey > New Opportunity > Framework > Achievement > Transformation); Perfect Webinar (Big Domino + 3 Secrets + The Stack + Close); The New Opportunity / New Vehicle; Breaking & rebuilding false-belief patterns
- **The Traffic Strategist** = Russell Brunson, *Traffic Secrets* - owns: Dream 100; Hook-bait for cold traffic; Pre-frame bridge for the unaware
- **The Story Brander** = Donald Miller, *Building a StoryBrand (StoryBrand SB7 / BrandScript)* - owns: SB7 BrandScript: Character > Problem (external/internal/philosophical) > Guide (empathy + authority) > Plan > Call to Action (direct + transitional) > Success > Failure/Stakes; StoryBrand homepage wireframe (header, stakes, value prop, guide, plan, explanatory, junk-drawer)
- **The Copy Closer** = Jim Edwards, *Copywriting Secrets* - owns: Headline formulas (4 U's: Useful, Urgent, Unique, Ultra-specific; How-To; Who-Else-Wants; curiosity gap); Fascination / bullet formulas; One-Sentence Persuasion (F-O-R-C-E); The 13-link sales chain; The Godfather offer / value stack
