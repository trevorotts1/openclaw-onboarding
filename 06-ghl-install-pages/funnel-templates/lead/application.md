# Application Funnel

**id:** `application`  |  **category:** Lead Funnels  |  **skill:** Skill 6  |  **length:** long-form

**Aliases:** High-Ticket Funnel, Phone Funnel, Backend Funnel, Three-Step Application Funnel

**Source (master library entry #7):** The Funnel Hacker's Cookbook, DotCom Secrets, Expert Secrets, Lead Funnels (named only)

## When To Use / Trigger Criteria

A 'take-away' sale where the prospect must APPLY to work with you. Used for high-ticket coaching, consulting, and any offer requiring a qualification phone call. Credibility video > qualification application > homework/booking, with the sale closed by phone using a setter-then-closer script.

**Goals it matches:**
- Sell a high-ticket offer
- Book qualified sales calls
- Pre-qualify / filter leads
- Set up a setter-to-closer phone close

**Keywords / request shapes:**
`high-ticket`, `coaching`, `consulting`, `book a call`, `application`, `apply to work with me`, `strategy session`, `qualify`, `done-for-you`

**Signals:**
- Offer is priced high enough to require a call
- You want only qualified prospects
- You want prospects to sell themselves through the form

**Do NOT use when:**
- Offer is low-ticket / self-serve (use Lead Magnet/SLO/Tripwire)
- You have no capacity to take sales calls

## Page Structure

### Page 1: Credibility / Case-Study VSL Page
*Purpose:* Establish authority and desire, then 'take away' by requiring an application (no price shown).

Key blocks:
- Case-study / credibility Video Sales Letter
- Big result proof
- Who this is (and isn't) for
- 'Apply Now' button

### Page 2: Application Form Page
*Purpose:* Qualify the prospect and get them to sell themselves.

Key blocks:
- Multi-field qualification form (situation, goals, ability/budget, 'why you')
- Commitment questions
- Submit routes to booking

### Page 3: Thank-You / Homework Page
*Purpose:* Bond with the prospect and book the call.

Key blocks:
- Calendar booking widget
- 'Homework' bonding videos
- What-to-expect on the call
- Prep instructions
- Scarcity on available call slots

## Copy Framework

**Primary persona (writes the copy):** The Expert Guide (Russell Brunson, Expert Secrets)

**Supporting personas:** The Funnel Architect (Russell Brunson, DotCom Secrets)

**Script:** Page-1 VSL = a condensed Perfect Webinar (Big Domino + proof + take-away CTA). Credibility story uses Star/Story/Solution. The call uses a setter-then-closer phone script.

**Copy structure / beats:**
- Authority story
- Big result / proof
- 'This isn't for everyone'
- Apply
- Qualify (prospect articulates their own desire)
- Book the call

**Notes:** The take-away (you must qualify) raises perceived value and inverts the chase. Form questions make prospects articulate their own desire before the closer ever calls.
