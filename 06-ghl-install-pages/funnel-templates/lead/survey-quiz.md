# Survey / Quiz Funnel

**id:** `survey-quiz`  |  **category:** Lead Funnels  |  **skill:** Skill 6  |  **length:** medium-form

**Aliases:** Quiz Funnel, Segmentation Funnel

**Source (master library entry #5):** The Funnel Hacker's Cookbook, DotCom Secrets, Expert Secrets

## When To Use / Trigger Criteria

A multi-question quiz used as a pre-frame bridge and segmentation tool before the main offer. Micro-commitments increase engagement; branching answers route visitors to different offer pages or email sequences based on their profile.

**Goals it matches:**
- Segment the audience by profile
- Increase engagement via micro-commitments
- Personalize the offer/result
- Route by answer to the right funnel/sequence
- Pre-frame the offer

**Keywords / request shapes:**
`quiz`, `survey`, `find out which`, `what's your type`, `personalized recommendation`, `segment`, `diagnostic`, `assessment`

**Signals:**
- Multiple buyer types / multiple paths exist
- You want personalization
- You want higher engagement than a flat opt-in
- You want data-driven routing

**Do NOT use when:**
- A single offer fits everyone (use Lead Magnet/Squeeze)
- The intent is open-ended research (use Ask Campaign)

## Page Structure

### Page 1: Quiz Start / Hook Page
*Purpose:* Promise a personalized result to earn the first click.

Key blocks:
- Outcome headline ('Discover your ___')
- Start button
- Social proof / number who've taken it
- What they'll learn

### Page 2: Multi-Step Quiz (3-20 questions)
*Purpose:* Stack micro-commitments and capture segmentation data.

Key blocks:
- One question per step
- Progress bar
- Conditional branching logic
- Optional email gate before showing results

### Page 3: Results + Routed Offer Page
*Purpose:* Deliver the personalized result and the matched CTA.

Key blocks:
- Personalized result copy
- Matched offer / lead magnet / sales page
- CTA
- Segment-specific follow-up trigger

## Copy Framework

**Primary persona (writes the copy):** The Funnel Architect (Russell Brunson, DotCom Secrets)

**Supporting personas:** The Story Brander (Donald Miller, Building a StoryBrand (StoryBrand SB7 / BrandScript))

**Script:** Survey funnel mechanics (DotCom Secrets) - a micro-commitment ladder - with the results page using the StoryBrand Plan > Call-to-Action for the matched segment.

**Copy structure / beats:**
- Curiosity hook ('Discover your ___')
- Effortless first question
- Escalating-relevance questions
- Email gate
- Personalized plan/offer per segment

**Notes:** The first question must be effortless to earn momentum; each answer is a tiny 'yes.' The segment determines both the offer and the email path.
