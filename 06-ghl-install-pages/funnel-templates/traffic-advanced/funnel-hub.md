## When To Use / Trigger Criteria
**Goals (request shapes that should match):**
- Give a personal brand or company a single authoritative home that looks like a real website to press/PR/partners while every internal link still routes into a front-end funnel.
- Capture 'shadow funnel' traffic — the organic research journey people take after seeing an ad or hearing the name.
- Organize all of a marketer's funnels, offers, blog posts and social content in one navigable place (a Value Ladder map).

**Keywords:** `funnel hub`, `brand hub`, `central website`, `home base`, `authority site`, `organize all my funnels`, `press/PR site`, `make me look legit`, `central place for everything`, `blog + social + funnels in one place`, `value ladder navigation`

**Signals:**
- Client already has multiple funnels/offers and needs a top-level brand home that ties them together.
- Client is running paid ads and wants to capture people who Google their name/brand afterward.
- Client needs a credible 'website' for media kits, partnerships, or guest appearances but does not want a dead-end brochure site.

**Anti-signals (do NOT match / route elsewhere):**
- Single-offer business with no other funnels yet — start with a squeeze/lead or front-end funnel first.
- Goal is a direct-response sale right now — use a Sales Letter / VSL / Webinar funnel, not the Hub.

## Page Structure
### 1. Hub Home (Brand Homepage)
*Purpose:* Pass the 5-second 'grunt test': instantly communicate who this is for, what they get, and the one primary next step. Routes the majority of traffic into the #1 front-end funnel.

*Key elements / blocks:*
- Brand logo + clean top navigation (Start Here, About, Content/Blog, Products, Contact)
- Hero: StoryBrand one-liner headline + subhead (problem -> plan -> success)
- Primary CTA button into the #1 front-end funnel (e.g. lead magnet / book / webinar)
- Trust bar: 'as seen in' / press logos / follower counts
- Value-ladder section: cards linking to each front-end funnel (not a generic shop)
- Featured content / recent blog teasers
- Email-capture band (secondary opt-in)

### 2. About / Manifesto (Attractive Character)
*Purpose:* Build the relationship and authority through origin story; convert curiosity into a follow + opt-in.

*Key elements / blocks:*
- Founder photo + Epiphany Bridge origin story
- Manifesto / mission statement (cause people can identify with)
- Attractive Character backstory, character flaws, and polarity
- Social-follow buttons (the shadow-funnel anchors)
- Inline opt-in to the signature lead magnet

### 3. Start Here / Value Ladder Map
*Purpose:* Orient a first-time visitor and route them to the correct front-end funnel for their stage.

*Key elements / blocks:*
- Short framing video or copy block (Hook-Story-Offer)
- Tiered cards: free lead magnet -> low-ticket -> core offer -> high-ticket
- Each card CTA links OUT to its own dedicated front-end funnel (never a shared cart)

### 4. Content / Blog Feed
*Purpose:* Feed organic + shadow-funnel search traffic, demonstrate expertise, and recycle every post into an opt-in.

*Key elements / blocks:*
- Blog RSS / post grid integration
- In-article + end-of-article CTAs into the relevant lead funnel
- Category navigation aligned to the Dream 100 topics
- Newsletter opt-in sidebar/footer

### 5. Press / Social Stream / Media
*Purpose:* Reinforce credibility for the shadow funnel — when someone researches the brand, owned properties dominate the result.

*Key elements / blocks:*
- Embedded social streams (Instagram/YouTube/podcast)
- Press mentions + media kit download (gated opt-in optional)
- Speaker / collaboration / podcast-guest CTA

### 6. Products / Funnels Index
*Purpose:* Window-shopping index where every product links into its own funnel rather than a generic product page.

*Key elements / blocks:*
- Product cards with image + one-line promise
- Each card routes to its dedicated VSL / sales-letter / order funnel
- No dead ends — every path ends in an opt-in or order step

## Copy Framework
**Primary persona:**
- **storybrand** — Donald Miller — StoryBrand persona (The Brand Clarifier): the SB7 framework — (1) a Character who (2) has a Problem, (3) meets a Guide, (4) who gives them a Plan, (5) and calls them to Action, (6) that ends in Success and (7) helps them avoid Failure — plus the one-liner and the grunt test.

**Supporting personas:**
- **strategist** — Russell Brunson — Traffic Secrets persona (The Traffic Strategist): Dream 100, Hook-Story-Offer for native content, the Funnel Hub + Shadow Funnel capture doctrine.
- **expert** — Russell Brunson — Expert Secrets persona (The Attractive Character / Expert): Epiphany Bridge story script, the Big Domino, false-belief patterns (vehicle/internal/external), Perfect Webinar, The Stack and Close, Hook-Story-Offer.

**Scripts:**
- StoryBrand SB7 BrandScript + one-liner for the Hub Home hero (clarity > cleverness; pass the grunt test).
- Epiphany Bridge (Expert Secrets) for the About/Manifesto origin story.
- Hook-Story-Offer (Traffic Secrets) on every value-ladder card and content CTA.

*Notes:* The Hub itself does not 'sell' — it clarifies and routes. Selling copy lives inside the front-end funnels each link points to. Keep one obvious primary CTA per page so the grunt test never fails.

## GHL Build Notes (Skill 6 + Skill 44)
**Pages (Skill 6):**
- Build as a GHL WEBSITE (multi-page with navigation/menu), NOT a linear GHL Funnel — Hubs need cross-page nav.
- Pages: Home, About/Manifesto, Start Here, Blog (GHL Blog module), Press/Social, Products index.
- Wire the top nav menu so every internal link target is a Funnel step, not another static page where possible.

**Skill 44 widgets (forms / calendars / surveys):**
- Skill 44 Form widget: hero/footer email opt-in (maps to a 'Hub Subscribers' tag).
- Skill 44 Form widget: media-kit / newsletter gated download on Press page.
- Optional Skill 44 Calendar widget on Contact for partnership/podcast bookings.

**Automations / workflows:**
- Workflow: new Hub subscriber -> Soap Opera Sequence (5 emails) -> Daily Seinfeld broadcast list.
- Retargeting audience pixel on all Hub pages to feed shadow-funnel recapture ads.

*Build notes:* Skill 6 provisions the GHL Website + blog; Skill 44 instantiates the opt-in/contact widgets. Set the GHL Website domain to the root brand domain so branded searches land here.

## Metrics To Calibrate

- Hub -> front-end funnel click-through rate
- Email opt-in rate per page
- Assisted conversions from Hub-touched visitors
