## When To Use / Trigger Criteria
**Goals (request shapes that should match):**
- Warm cold social audiences organically BEFORE they ever hit the main funnel.
- Pre-frame the offer so that by the time someone clicks the bridge CTA, they already trust the Attractive Character.
- Draw people in natively on each platform (described as equally important as the funnel itself).

**Keywords:** `pre-frame`, `pre-funnel`, `warm up audience`, `social content`, `organic`, `stories`, `reels`, `posts`, `pinterest`, `facebook live`, `content that feeds the funnel`, `attractive character content`, `bridge CTA`

**Signals:**
- Client has (or is building) an organic social presence and wants it to feed funnels, not just collect likes.
- Cold audiences bounce off the funnel — they need a relationship/pre-frame first.
- Client is working a Dream 100 / organic strategy.

**Anti-signals (do NOT match / route elsewhere):**
- Client wants immediate paid-traffic conversions only — pair this with, but don't substitute it for, a front-end funnel.
- No social presence and no willingness to publish consistently.

## Page Structure
### 1. Content Unit: Instagram/TikTok Stories & Reels
*Purpose:* High-frequency, native, behind-the-scenes + value content that builds the Attractive Character daily.

*Key elements / blocks:*
- Hook in first 1-2 seconds (Edwards scroll-stopper)
- Mini Epiphany-Bridge story or quick value
- Bridge CTA: link sticker / 'link in bio' to the squeeze/lead-magnet funnel

### 2. Content Unit: Facebook Posts & Lives
*Purpose:* Longer-form story + teaching that pre-frames the core offer's big domino belief.

*Key elements / blocks:*
- Hook-Story-Offer post structure
- Live teaching that demos the result
- Comment/DM trigger -> auto-DM the funnel link

### 3. Content Unit: Pinterest Pins / Blog
*Purpose:* Searchable, evergreen pins/articles that pull new cold audiences in over time.

*Key elements / blocks:*
- Keyword/curiosity pin titles
- Pin/article links to a lead-magnet funnel
- Value-first, no hard sell

### 4. Bridge Destination: Squeeze / Lead-Magnet Funnel
*Purpose:* Where every content unit's CTA lands — convert the warmed follower into an email lead.

*Key elements / blocks:*
- Curiosity squeeze headline matching the content promise
- Single opt-in form
- Routes into the value ladder + follow-up
