KIE.AI SETUP - INSTALLATION GUIDE
══════════════════════════════════════════════════════════════════

This guide walks you through creating a KIE.ai account, getting your API key,
adding credits, and connecting KIE.ai to your OpenClaw setup. KIE.ai is a
service that lets your AI agent generate images and videos using models like
Nano Banana Pro, Kling, Sora, and VEO.

Think of it like this: KIE.ai is the "art studio" your AI uses to create
pictures and videos on your behalf.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 1: CREATE YOUR KIE.AI ACCOUNT
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Open your web browser (Chrome, Safari, Edge, or Firefox).

2. Go to this website: https://kie.ai

3. You will see the KIE.ai homepage. Look for a "Sign Up" or "Get Started"
   button. It is usually in the top right corner of the page.

4. Click "Sign Up" and create your account. You can sign up with:
   - Your email address and a password
   - Or your Google account (if you prefer, click "Continue with Google")

5. If you signed up with email, check your inbox for a verification email
   from KIE.ai. Click the link in that email to verify your account.

6. After verifying, log in to KIE.ai. You should see your dashboard.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 2: ADD CREDITS TO YOUR ACCOUNT
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

KIE.ai is a pay-as-you-go service. That means you buy credits, and each time
your AI generates an image or video, it uses some of those credits.

Here is a rough idea of what things cost:
- One image (Nano Banana Pro): about $0.09
- One video clip (VEO 3.1 Fast): about $0.40
- One video clip (Sora 2): about $0.15

7. From your KIE.ai dashboard, look for a "Pricing" or "Credits" or "Billing"
   link. It may be in the top navigation bar or in a sidebar menu.

8. You can also go directly to: https://kie.ai/pricing

9. Choose how many credits you want to add. If you are just getting started,
   $10 to $20 is a good amount to test with. You can always add more later.

10. Enter your payment information (credit card or other accepted method)
    and complete the purchase.

11. After payment, your credit balance should update on your dashboard.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 3: GENERATE YOUR API KEY
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

An API key is like a password that lets your AI agent talk to KIE.ai on your
behalf. Your agent needs this key to generate images and videos for you.

12. Go to: https://kie.ai/api-key
    (You can also find this by clicking on "API Settings" or "API Keys" in
    your KIE.ai account settings.)

13. Click "Create API Key" or "Generate New Key" (the exact button text may
    vary slightly).

14. IMPORTANT: The API key will be shown to you only ONCE. It is a long
    string of letters and numbers. Copy it immediately.

15. Paste the key somewhere safe for now (a text file, a note, etc.). You
    will need it in the next step.

    If you accidentally close the page without copying the key, do not worry.
    Just delete that key and create a new one.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 4: ADD YOUR API KEY TO OPENCLAW
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Now you need to tell your OpenClaw agent where to find the KIE.ai API key.

16. Open your terminal (the command line application on your computer).
    - On Mac: Open the app called "Terminal" (you can search for it in
      Spotlight by pressing Command + Space and typing "Terminal")
    - On Windows: Open "Command Prompt" or "PowerShell"

17. Type this command and press Enter:

    nano ~/.openclaw/openclaw.json

    This opens the OpenClaw configuration file in a text editor called nano.

18. Look for a section that says "env". If it exists, add the KIE_API_KEY
    line inside the "vars" section. If it does not exist, add the whole
    block. It should look like this:

    {
      "env": {
        "vars": {
          "KIE_API_KEY": "paste-your-actual-api-key-here"
        }
      }
    }

    Replace "paste-your-actual-api-key-here" with the actual API key you
    copied in Step 3.

19. Save the file:
    - If you are using nano: Press Control + O, then press Enter, then
      press Control + X to exit
    - If you are using another editor: Save and close normally

20. Restart your OpenClaw gateway so it picks up the new key:

    openclaw gateway restart

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 5: SET UP WEBHOOK (OPTIONAL)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Webhooks let KIE.ai notify your agent when a job is finished, instead of
your agent having to keep checking over and over. This is optional but
recommended if you plan to use KIE.ai heavily.

21. Go to: https://kie.ai/settings

22. Look for a "Webhook" section.

23. If you want to use webhooks, you will find a "webhookHmacKey" on that
    page. Copy it and save it. Your agent will use it to verify that
    notifications are really coming from KIE.ai.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 6: VERIFY EVERYTHING WORKS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Let's make sure your setup is working correctly. Run these tests one at a
time in your terminal.

TEST 1: Check that your API key is loaded

24. In your terminal, type this and press Enter:

    echo $KIE_API_KEY | head -c 10

    You should see the first 10 characters of your API key printed out.
    If you see nothing (a blank line), the key was not saved correctly.
    Go back to Step 4 and try again.

TEST 2: Check your credit balance

25. In your terminal, type this and press Enter:

    curl -s -H "Authorization: Bearer $KIE_API_KEY" \
      "https://api.kie.ai/api/v1/chat/credit"

    You should see something like this:
    {"code":200,"msg":"success","data":100}

    The number after "data" is your credit balance. If you see a 401 error,
    your API key is wrong. Go back to Step 3 and create a new one.

TEST 3: Generate a test image

26. In your terminal, type this and press Enter:

    curl -s -X POST "https://api.kie.ai/api/v1/jobs/createTask" \
      -H "Authorization: Bearer $KIE_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{
        "model": "nano-banana-pro",
        "input": {
          "prompt": "A simple red circle on white background",
          "aspect_ratio": "1:1",
          "resolution": "1K"
        }
      }'

    You should see a response that includes a "taskId" and a status.
    If you get an error, double check your API key and credit balance.

TEST 4: Check on the test image

27. Wait about 10 to 15 seconds, then type this command. Replace
    YOUR_TASK_ID with the actual taskId you got from Test 3:

    curl -s "https://api.kie.ai/api/v1/jobs/recordInfo?taskId=YOUR_TASK_ID" \
      -H "Authorization: Bearer $KIE_API_KEY"

    When it is done, you will see a status of "success" and a list of URLs
    where your generated image can be downloaded.
