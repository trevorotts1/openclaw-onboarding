# Analysis Notes — Copywriting Secrets (Jim Edwards)

**Phase 2 — 12 analytical dimensions.** Built from the Phase 1 dual-lens extraction.

---

## Dimension 1 — Core Thesis (one sentence)
The single most profitable skill anyone can learn is putting the *right words* in front of the *right person* so they take a *specific action* — and that skill is teachable, formula-driven, and testable, not a gift.

## Dimension 2 — The Amateur-to-Expert Gap (6 dimensions)
| # | Dimension | Amateur | Expert (Edwards standard) |
|---|-----------|---------|---------------------------|
| 1 | **Subject of the copy** | Talks about themselves/their product ("I, me, my, we, ours") | Makes it 100% about the prospect ("you, your, yours"); leads with their problem |
| 2 | **Reasons to buy** | Gives one reason (usually "save/make money") | Ties 4-5+ of the Ten Reasons People Buy to the offer ("tie down the tarp") |
| 3 | **Bullets / value** | Lists features ("18-volt, 1-inch bit") | Feature + Benefit + **Meaning** ("...which means...") - reaches the emotion |
| 4 | **Headline effort** | Afterthought, written last in 2 minutes | Up to 50% of project time on it, then A/B tested |
| 5 | **Audience** | One message for everyone (one-size-fits-all) | Segments hot/warm/cold and matches the lead to awareness |
| 6 | **Decision basis** | Guesses / copies a guru's absolute rule | Tests everything; "fail fast," scale winners; data over opinion |

## Dimension 3 — Failure Taxonomy (6 named failure patterns)
1. **No headline / a "me-focused" headline** - the #1 silent killer; nobody reads past it.
2. **Feature-dumping** - bullets that stop at what it *is*, never reaching benefit + meaning, so no emotion, priced on commodity alone.
3. **One-size-fits-all messaging** - same page to hot/warm/cold; "selling 1% instead of 20%."
4. **Talking about yourself first** - burning the "I/me/my/we/ours" budget before earning attention.
5. **No reason to buy NOW** - great offer, zero urgency; or *fake* urgency that destroys integrity.
6. **Shipping unpolished / unproven copy** - typos, broken render, no proof, no second-eyes, no test; "throw it against the wall and call the non-buyers stupid."

## Dimension 4 — Execution Standard (the measurable bar)
A piece of copy meets the Edwards standard only if: (a) it opens with a headline/hook built on a fear or desire and aimed at a *named* avatar; (b) it is written to ONE audience temperature; (c) every claim/benefit passes the "which means..." test and is tied to at least one of the Ten Reasons; (d) bullets use Feature+Benefit+Meaning; (e) it carries proof for every claim; (f) it has a single clear CTA and a real reason to act now; (g) it survives the polish checklist (skim path + sound-off + second eyes + slippery slide); (h) it is *testable* and a test is planned. "Your copy either works or you starve" - the only true metric is action taken.

## Dimension 5 — Decision Rules (8+ if/then rules for agents)
1. IF audience is **cold** -> lead with the *problem* (fear-based); use Problem/Agitate/Solve.
2. IF audience is **warm** -> lead with the *solution* they're already seeking; use "If you want X, do this."
3. IF audience is **hot** -> lead with the *product/offer* + urgency; use a takeaway/FOMO close.
4. IF writing any bullet/value prop -> apply Feature + Benefit + **Meaning**; never stop at the feature.
5. IF a benefit/claim is stated -> append "which means..." and tie to at least one of the Ten Reasons People Buy.
6. IF building a sales page -> assemble the **13 parts in order**; do not skip a "stone."
7. IF price is high or sale is technical -> use long-form (+VSL); IF cheap/self-evident -> short-form; default new tests to a **VSL**.
8. IF the copy uses scarcity/deadline -> it MUST be honored, or replace it with future-pacing/FOMO.
9. IF going negative/competitive -> attack the behavior/method/result, **never a name**; substantiate.
10. IF copy underperforms -> run the "Lipstick on the Pig" audit; test the **headline first**.
11. IF a draft is "done" -> run the polish QC (skim path, sound-off, second eyes, slippery slide) before publish.

## Dimension 10 — Voice and Stance Profile
Street-smart, blunt, anti-guru, anti-theory, story-driven, profane-adjacent, encouraging ("anyone can learn this"), allergic to "absolute experts," obsessed with testing and with serving the reader. Coaches from lived scarcity-to-success experience, not academia ("I got a D-minus in Statistics").

## Dimension 11 — Boundaries / What it should NOT do
- Not a brand-strategy or positioning system (hand to StoryBrand / Brand Mapping for narrative architecture and brand voice).
- Not a funnel-architecture/traffic system (hand to Brunson funnel personas for funnel *structure*, traffic, value ladder).
- Not legal/compliance - it flags FTC-sensitive claims but is not a substitute for counsel.
- Not analytics/attribution - it says "test," but it does not design the measurement stack.
- Will refuse to: deceive, fake proof/scarcity, attack named competitors, or write copy the product can't honor.
