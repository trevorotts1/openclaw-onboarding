# Analysis Notes — Building a StoryBrand (Donald Miller)

12 analytical dimensions.

## 1. Amateur-to-Expert Gap (6 dimensions)
| Dimension | Amateur | Expert |
|---|---|---|
| Hero of the story | The brand | The customer |
| Problem framing | External only | External + Internal + Philosophical |
| Trust | Bragging / silence | Empathy + light authority (<=3 proofs) |
| Call to action | Soft or missing | One bold, repeated direct CTA + transitional |
| Word economy | Dense paragraphs | "Morse code," scannable, cut in half |
| Net effect | Comprehensive & confusing | Clear & story-shaped; customer wins |

## 2. Failure Taxonomy
1. Brand-as-hero (the fatal mistake — "you will die").
2. External-only selling (ignoring the internal feeling people actually buy).
3. Whisper/absent CTAs (underselling — almost no one oversells).
4. Wall-of-words above the fold (people scan, don't read).
5. No stakes / no story (or the opposite: fearmongering).
6. Clever-over-clear (failing the grunt test for the sake of wit).
7. "Newsletter signup" lead capture (no value = spam in the reader's mind).
8. Boiling the ocean (not narrowing to one BrandScript / one-liner).

## 3. Execution Standard
A deliverable is done when a stranger, in <=5 seconds, can repeat what's offered, why it helps them, and the next step — and *wants* to take it. Every line traces to a BrandScript module. Brief, scannable, customer-as-hero, empathy+authority, one repeated CTA, stakes as a pinch, aspirational payoff.

## 4. Decision Rules (8+)
1. If hero != customer -> rewrite (customer is the hero, always).
2. If grunt test fails in 5s -> simplify before anything else.
3. If only external problem named -> add the internal feeling.
4. If trust gap -> add empathy + <=3 authority proofs (never bragging).
5. If reader hovers/won't commit -> add a 3-4 step process plan.
6. If no email capture -> build a high-value lead generator (transitional CTA).
7. If no urgency -> add stakes (a pinch of failure), never fearmonger.
8. If copy is dense -> reduce to Morse code, bullets, "read more."
9. If many products -> find an umbrella message, split onto sub-pages w/ own BrandScripts.
10. If building a funnel -> follow the Roadmap order (one-liner->site->lead gen->drip->stories->referrals).

## 5. Department / Role Application Map
- Marketing: Brand Messaging Strategist, Content Strategist, Email-Campaign Strategist.
- Creative: value props, ad copy, brand voice.
- Web Development / Funnel: Website Copywriter, Funnel Copywriter, Funnel Builder Specialist (copy -> grunt-test -> Skill 6 build).
- Sales / CRM: CTA logic (direct vs transitional), nurture->sales sequences, lead-magnet specs.
- Leadership: founder one-liner, internal narrative/culture ("people want to be taken somewhere").

## 8. Evidence / Case Examples (real, from the book)
Apple, Tidal (Jay Z — cautionary), The King's Speech, Star Wars/Yoda, high-end resort ("luxury and rest"), Squarespace, Local Honey ("We sell clothes. We do hair."), Gerber "Hello Trouble," Dave Ramsey, Allstate "Mayhem" / Project Share Aware, Crest Hill dog kennel (nurture email), StoryBrand's own "5 Things Your Website Should Include" lead generator (>40k downloads -> $2M).

## 9. Two-Sided Mapping
COACHING side = grunt-test diagnosis, hero/guide reframe, "what do you want," pick one BrandScript + one-liner, weekly commitment. LEADERSHIP side = grunt-test bar, BrandScript-as-source-of-truth, Five Things rubric, roadmap funnel SOP, lead-gen + email-drip formulas, failure-pattern rejection.

## 10. Boundaries / Hand-offs
Authors the MESSAGE, not the page build (-> Skill 6) or the offer economics (-> Hormozi/offer personas) or the visual identity (-> design personas). StoryBrand owns clarity of *language*.

## 11. Quantitative Anchors
Grunt test <=5s; ~3 testimonials; lead-gen PDF ~3 pages; email drip 3 nurture : 1 sales; 20% email open rate is standard; loss aversion 2-3x; "cut half the words"; one obvious (repeated) button.

## 12. Risk / Misuse Notes
Do not weaponize stakes into fearmongering (villain must be real; salt, not the meal). Do not strip so much that the message loses substance — clarity != thinness; the goal is *robust, brand-building clarity*, which is exactly the upgrade BlackCEO wants for website + funnel copy. Do not confuse comprehensiveness with clarity.
