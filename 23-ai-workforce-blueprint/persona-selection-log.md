# Persona Selection Log

Every task dispatch MUST produce a log entry here. No exceptions.

## Format

```
[date] [task-id] [candidates-considered] [selected-persona] [layer-3-reason] [layer-4-reason] [layer-5-reason]
```

## Columns

| Field | Description |
|-------|-------------|
| date | ISO date (YYYY-MM-DD) |
| task-id | Task identifier |
| candidates-considered | Comma-separated list of top persona candidates evaluated |
| selected-persona | The persona chosen for this task |
| layer-3-reason | Why this persona aligns with company goals/KPIs |
| layer-4-reason | Why this persona aligns with department goals/KPIs |
| layer-5-reason | Why this persona fits this specific task |
