# ARCHIVED — DO NOT RUN — Skill 33 installation guide

> **There is no installation procedure for this skill any more. Do not run it.**
>
> **Successor: Skill 23 — AI Workforce Blueprint (`23-ai-workforce-blueprint/`).**

The procedure this file used to carry created seventeen fixed department
workspaces and added seventeen entries to `agents.list[]` in
`~/.openclaw/openclaw.json`. Skill 23 now builds the workforce from the owner
interview. Running the old procedure against a box Skill 23 has already built
produces duplicate and conflicting registry entries and orphaned workspaces.

`install.sh` in this folder refuses to run and exits non-zero.

To install the live skill, follow `23-ai-workforce-blueprint/INSTALL.md`.

See `SKILL.md` and `ARCHIVED.md` in this folder for the capability map.
