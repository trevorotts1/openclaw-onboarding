# ARCHIVED — DO NOT RUN — Skill 34: Intelligent Workspace Staffing

> **This skill is archived. Do not run its installer. Do not follow it as a procedure.**
>
> **Successor: Skill 23 — AI Workforce Blueprint (`23-ai-workforce-blueprint/`).**

Skill 34 was a standalone classification pass that ran after a workforce was
already built: permanent-versus-on-call classification, persona alignment and a
package tier, written back into each role's `IDENTITY.md`. All three now happen
inline inside the Skill 23 build. Skill 34 no longer extends any skill and no
longer depends on any skill; the statements to that effect were removed with this
tombstone, because both of the skills they named have changed shape underneath.

## Why you must not run this

The interview this skill used to run asked six questions per department across a
fixed set of seventeen departments. Skill 23 does not build seventeen fixed
departments. Running this installer, or following the procedure it used to
document, stamps `specialist_type`, `assigned_persona` and `package` values onto
roles that Skill 23 has already classified, and creates specialist workspaces and
sub-agent templates that no Skill 23 build state knows about.

`install.sh` in this folder refuses to run and exits non-zero. That refusal is
deliberate. Do not remove it.

## Where each capability went

| Old Skill 34 capability | Where it lives now |
