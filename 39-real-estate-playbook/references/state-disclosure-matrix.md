# State Disclosure Pointer Matrix (50 states + DC)

**This is a CONVENIENCE POINTER matrix, NOT legal advice.** It points to WHERE
the authoritative disclosure requirements live per state; it does not state the
law. The disclosure decision always escalates to the licensed agent/broker, and
the authoritative source named below governs (statutes change; this matrix may
lag). Each row carries a `matrix_pointer_id` used in the `disclosure_surfaced`
event.

## How to use

1. Resolve the property state (from geocode `state` or the operator).
2. Find the row; surface the "typical disclosures" + "authoritative source".
3. Escalate the decision. Do not fill or sign forms.

## Cross-cutting (always confirm with the authoritative source)

- **Federal lead-based paint disclosure** applies to most pre-1978 housing
  nationwide (separate from any state form).
- **Pre-foreclosure / foreclosure-consultant** rules vary by state and may add
  cooling-off + disclosure obligations — surface the state pointer and escalate
  before any distressed-owner solicitation.
