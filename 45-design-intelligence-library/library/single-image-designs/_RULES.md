# CATEGORY RULES — Single Image Designs (SI-)
**The general-purpose category: standalone visuals that aren't tied to a specific ad/cover/banner format — hero images, blog headers, thumbnails, slide imagery, concept art, any one-off.**

## Formats & aspect ratios
- No fixed format. The card records its native ratio; the operator may request any ratio the target model supports (see MODEL-SPECS aspect table).
- When re-ratioing a style (e.g., 1:1 source → 16:9 output), re-map the 9-zone layout: state in the prompt where elements move, don't let the model guess.

## Hard rules
- NEVER text over faces (when text present).
- SI cards are the most reused across categories — write them with maximum content-separation rigor (Golden Rule), since they'll be transferred furthest.
- If an SI style is reused 3+ times for one format (e.g., always as YouTube thumbnails), clone it into the proper category with its own ID and format rules.
- Skin-tone hard rule applies to all people imagery: rich, warm, dimensional deep skin tones.

## Model routing
- Follow MODEL-SPECS routing table by task. No category override.
