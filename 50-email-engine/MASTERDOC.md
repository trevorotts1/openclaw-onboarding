## A. THE 13 EMAIL FRAMEWORKS (the reusable IP)

Framework is one of these 13 canonical ids. Where a structured `sections[]` is supplied it must equal the framework's declared part count (`AF-EMAIL-FRAMEWORK-INCOMPLETE`); an unknown framework is `AF-EMAIL-FRAMEWORK-UNKNOWN`.

1. **Features-to-Benefit** — link each product feature to a concrete customer benefit ("This [FEATURE] means you'll [BENEFIT], which allows you to [DEEPER BENEFIT]"). Structure: **Intro -> feature->benefit pairs -> summary -> CTA.**
2. **PAS (Problem-Agitate-Solution)** — identify a resonant **Problem** -> **Agitate** (deepen it, show it worsening) -> present the **Solution** + CTA. Pain-avoidance driven. **(3 parts)**
3. **AIDA (Attention-Interest-Desire-Action)** — grab **Attention** -> stimulate **Interest** -> build **Desire** (unique benefits) -> strong clear **Action** (CTA). **(4 parts)**
4. **P.A.S.T.O.R. V1 (Solutions Version)** — **Problem -> Amplify** (+ aspirations) **-> Story & Solution -> Transformation & Testimony -> Offer -> Response.** Ray Edwards formula; comprehensive, trust-building. **(6 parts)**
5. **P.A.S.T.O.R. V2 (Story Version)** — the same **6 parts**, narrative-led: position the product as the solution INSIDE a story; testimonials as social proof; emotional connection primary. **(6 parts)**
6. **Million Dollar Sales** — **12 beats:** Attention Grabber -> Intention Call-Out -> Disqualifier -> Credibility -> Context -> CTA#1 -> Benefits -> Testimonial -> CTA#2 -> Final Intention Call-Out -> Final CTA#3 -> Humorous P.S. Relationship + multiple response chances. **(12 parts)**
7. **Before-After-Bridge (BAB)** — **Before** (current pain) -> **After** (improved future) -> **Bridge** (your solution is the path). Contrast principle. **(3 parts)**
8. **The 6 W's** — **Who, What, Why, When, Where, hoW** — answer all six; comprehensive; ideal for methodical decision-makers. **(6 parts)**
9. **Star-Chain-Hook** — **Star** (big attention-grabbing idea) -> **Chain** (facts / sources / reasons / benefits that build credibility) -> **Hook** (CTA). The chain (proof points) is the key element. **(3 parts)**
10. **Star-Story-Solution** — **Star** (main character) -> **Story** (character faces the reader's problem) -> **Solution** (how the character wins). Narrative transportation. **(3 parts)**
11. **The 3-B Plan** — **Brevity, Bluntness, Basic.** Short, blunt, simple; get to the point / incentive fast. Best for busy / spontaneous buyers. **Under ~150 words** (`AF-EMAIL-WORDBAND`).
12. **ACCA (Awareness-Comprehension-Conviction-Action)** — present the situation (**Awareness**) -> describe the impact (**Comprehension**) -> create desire / conviction that your solution fixes it (**Conviction**) -> ask for a response (**Action**). Like AIDA with clarity + certainty emphasis. **(4 parts)**
13. **Cold-Outreach Variants** — the cold-traffic variants of **Star-Chain-Hook** and **Star-Story-Solution** (see #9 / #10): first-touch, no prior relationship, reply-goal, short.

## B. THE BUYER-TYPE -> EMAIL# -> FRAMEWORK MAP (Convert & Flow + High-Ticket, 12-email)

The same mapping governs the Convert & Flow master 12-email sequence and the High-Ticket Appointment 12-email sequence. A slot's framework or buyer band that does not match is `AF-EMAIL-BUYERTYPE-MAP`.

- **Spontaneous (Emails 1-2):** E1 = **3-B Plan**; E2 = **Star-Chain-Hook**.
- **Methodical (Emails 3-6):** E3 = **Features-to-Benefit**; E4 = **6 W's**; E5 = **ACCA**; E6 = **PASTOR (Solutions)**.
- **Humanistic (Emails 7-9):** E7 = **PASTOR (Story)**; E8 = **Star-Story-Solution**; E9 = **PAS**.
- **Competitive (Emails 10-12):** E10 = **AIDA**; E11 = **Million Dollar Sales**; E12 = **Before-After-Bridge**.

**Every email must stay in harmony with all previous emails (except #1)** — one voice, one offer, one escalating argument (`AF-EMAIL-SEQUENCE-DISCONTINUITY` in doctrine; continuity is the retention mechanism).

**The four buyer types:**
- **Spontaneous** — fast, gut-feel, novelty-driven, low-friction; wants the point fast (3-B Plan, Star-Chain-Hook).
- **Methodical** — analytical, evidence- and detail-driven, risk-averse, deliberate; wants specifics and guarantees (Features-to-Benefit, 6 W's, ACCA, PASTOR-Solutions).
- **Humanistic** — relationship- and story-driven, people-first, belonging; moves on warmth (PASTOR-Story, Star-Story-Solution, PAS).
- **Competitive** — results-driven, status / achievement / edge, skeptical of fluff; moves on proof and exclusivity (AIDA, Million Dollar Sales, BAB).

## D. THE 4 SEQUENCE OBJECTIVES

Every email targets **exactly ONE** objective (`AF-EMAIL-OBJECTIVE-INVALID` otherwise). Each objective has tactical variants for Spontaneous / Methodical / Humanistic / Competitive buyers.

1. **Promotional** — warm audience; scarcity / FOMO, exclusivity / belonging, social proof, anticipation / curiosity, value-driven desire.
2. **Abandoned Cart** — loss aversion, cognitive-dissonance resolution, Zeigarnik effect (unfinished business), trust / risk reduction, reciprocity; **the first email sends within 1-3 hours.**
3. **Upsell** — commitment / consistency, perceived-value multiplication, status / aspiration, fear-of-suboptimal, convenience / efficiency.
4. **Downsell** — risk reduction, incremental commitment (foot-in-door), value recalibration, future-opportunity preservation, face-saving.
