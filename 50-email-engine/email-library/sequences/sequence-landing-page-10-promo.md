## Style / Definition
A **10-email promotional campaign tied directly to the Signature Landing Page.** Each email runs a specific framework in a fixed order, mines specific landing-page sections for its raw material, and follows a strict format contract (word count, dual subject lines, dual preview lines, labeled body, founder signature).

**Trigger rule:** After the Downsell Page is written, immediately append this exact prompt: *"Would you like me to start writing 10 promo emails for your landing page."* — If YES, write all 10 using the framework map below. **Follow exactly; no deviation.**

## Purpose & Intention
The landing page already contains the campaign — the pain points (Section 6, "Big Bold Who"), the mission (Section 5, "Big Bold Why"), and the offer details (Sections 8–10) are pre-validated persuasion assets. This sequence converts that single page into ten distinct attacks on the same objective, each from a different psychological angle: three pain-point-led PASTOR emails to open, then a rotation through seven more frameworks so no two sends feel alike. Framework variety is the retention mechanism — the list stays fresh through ten promotional emails because each one *reads* like a different email, while every one points at the same door.

**PASTOR rule for E1–E3:** position the product/service/event as the solution; each email owns a DIFFERENT problem (identify the problem, amplify the consequences of not solving it), tell a story, include testimonials, weave in a refined version of Section 5 ("Big Bold Why"), then ask for a response. **≥3 CTAs per email** — near the beginning, natural middle, near the end.

## How To Write It (sequence rules — all 10 emails)
1. **Word count: 150–300 words** per email — EXCEPT Email #7 (3-B Plan), which is deliberately shorter (<150).
2. **2 subject lines per email** (A/B split test), labeled.
3. **2 preview lines per email** (A/B), labeled. **The second subject/preview set must be prolific + disruptive** — designed to shock and grab attention. Set A plays it straight; Set B breaks the pattern (bold claim, unexpected frame, controlled shock — never crude, never off-brand-values).
4. **Label the body "Email Body:"** and **label each email by the framework used.**
5. **Continue the tone/style established for the brand** — the campaign should read as the landing page's voice arriving in the inbox.
6. **Use the founder's ACTUAL name in the closing signature** — no placeholder ships. (In this library's brand-agnostic examples, `[Founder Name]` marks the slot; the generation engine must substitute the real name.)
7. **Mine before you write.** Extract pain points 1–3, the refined Big Bold Why, and the Section 8–10 offer details BEFORE drafting; every email should be traceable to its landing-page source.
8. **Sequence hygiene:** each email stands alone (readers miss sends) but never contradicts a previous one; vary CTAs' phrasing while keeping one destination.

## Use Cases
- The standard follow-through after any Signature Landing Page build (triggered after the Downsell Page)
- Launch campaigns for the offer the landing page sells
- Evergreen nurture: the 10 emails re-deployed as an automated post-opt-in sequence
- Re-promotion windows (doors-reopen campaigns) using the same map with refreshed examples
