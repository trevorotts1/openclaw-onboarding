# Frame — The Rulebook

> Derived from the Rulebook archetype (a leadership coach for professional women). The teaching
> spine is a set of **numbered, non-negotiable Rules**. Every frame runs the identical 4-phase
> skeleton; The Rulebook only changes the teaching devices, refrains, and the close. The
> non-negotiables below (slide ranges, ≥100 floor, ≤2 case studies, 3–7 steps, suggested image per
> slide, N.E.E.I.T./4-Quadrant, Phase-3-no-pitch, central + section hooks) are enforced by
> `prove_sp_structure.py` / `prove_sp_no_pitch.py`, not by this template.

- **signature_frame id:** `rulebook`
- **Teaching unit:** a **Rule** (3–7 total, one per teaching step)
- **Per-Rule shape:** teaching → **affirmation** → a **3-step action plan**
- **Signature devices:** hashtag refrains; a riddle ("who am I?" reveal); a recap-of-all-Rules slide; a teased bonus **"Rule #8???"**
- **Central hook:** the deck's one repeated chorus line; **section hooks:** one per phase, laddering to it

## Phase 1 — Avatar Section (slides 1–11)
- N.E.E.I.T. + 4 Quadrants. Open inside the AUDIENCE's story, never with a self-introduction.
- Devices: plant 1–2 refrain hashtags the audience will repeat (e.g. an authenticity refrain, a reclaim-power refrain).
- Beats: current-state pain ("Sometimes I wonder if I'm enough"), the overachiever's trap, a draining relationship/pattern, then the aspirational avatar.
- Tone ladder: empathetic → relatable/passionate → hopeful/visionary → motivating/celebratory. Section-hook slide for Phase 1.

## Phase 2 — Signature Story (slides 12–24)
- N.E.E.I.T. + 4 Quadrants. Relatability → vulnerability → pain-into-purpose → legacy.
- Rulebook flavor: the "I left an established profession to bet on myself" WHY; a borrowed-authority quote; an "I'm Worth the Risk" beat.
- Close the section on a gripping "why". Section-hook slide for Phase 2.

## Phase 3 — Transformational Teaching (slides 25–60) — NO PITCHING
- The teaching is **the Rules**: 3–7 numbered Rules, each a teaching step.
- Each Rule slide-group: Rule title (chapter-style) → teaching (mix questions, stories, one-liners, definitions, formulas, media moments) → an **affirmation** → a **3-step action plan**.
- Insert the riddle device once as an attention reset. Tag **≤2 CASE_STUDY** slides across the deck (band 1–2).
- Near the end of Phase 3: a **recap slide listing all Rules**, then the teased **"Rule #8???"**.
- Final teaching step = the bridge: summarize value + social proof + "what comes next" — **no price, no product name, no enroll CTA** (that is Phase 4).

## Phase 4 — Purpose Pitch (slides 61–100)
- N.E.E.I.T. + 4 Quadrants. Purpose pitch, not profit pitch — make the **purpose-pitch-vs-profit-pitch** contrast explicit on its own slides.
- Introduce the offer (the exact q7 product name), benefit bullets, proof, honest scarcity, guarantee, repeated conviction CTAs.
- **Signature close:** a **roll-call of iconic figures the audience admires** ending "…AND YOU!" with the offer URL.

## Refrain & tone policy
- 2–4 recurring hashtag refrains max; reuse them as callbacks, don't invent new ones every slide.
- Every phase carries its labeled name + purpose slide (Directive 10) before its content.

## Visual identity & suggested-image seed craft (Directive 4)
Every slide's `suggested_image` is the AUTHORING SEED the Prompt Author expands into the
9,000–18,000-char rich prompt — it never replaces that floor, but a strong seed is what makes the
expansion land. Write every seed with the six anchors: **[archetype A1–A5] + subject & emotion +
setting/world + light + palette anchor (from the locked STYLE BLOCK) + composition zone for the
baked copy**. Never put slide copy in the seed (copy is baked at prompt time); describe the PICTURE.

**The Rulebook's visual world — bold editorial authority:**
- **Controlling visual motif:** the Rules are a SERIES — every Rule-title slide uses the same
  editorial plate system (type-dominant archetype A4: oversized rule number + designed display
  type on a consistent grid) so Rules 1..7 read as chapters of one book. State the shared grid in
  every Rule-title seed; only the number and accent treatment change.
- **Affirmation + 3-step-plan slides:** warm human counterpoint to the bold plates — close-framed,
  camera-facing or near-camera subjects, soft directional light; the 3 steps as a clean designed
  list zone, never a cluttered diagram.
- **Avatar slides (1–11):** the audience's current state in their own world — real settings,
  emotionally honest expressions; pain beats in muted palette, the take-back-power refrain beats
  in rising brand-accent energy.
- **Story slides (12–24):** the leap narrative — the old profession's world giving way to the
  new one; the borrowed-authority quote as a designed-typography plate; "I'm Worth the Risk" as a
  confident, light-forward portrait beat.
- **Riddle + recap + "Rule #8???" tease:** attention-reset plates — spare, high-contrast,
  type-dominant; the recap lists all Rules in the series grid; the tease withholds (a deliberate
  visual gap or veiled element).
- **Roll-call close:** celebratory crescendo — a sequence of iconic-energy portrait frames landing
  on "…AND YOU!" as a mirror moment: the subject facing the audience straight-on, brightest light
  of the deck, offer URL as designed type.
- **Series discipline:** vary archetypes between adjacent slides (anti-template SOP-04) while
  holding palette + grid constant; portraits/likeness and the logo ride image-to-image references
  per SOP-IMG-01 — the seed notes WHERE they sit, never re-describes the mark.
