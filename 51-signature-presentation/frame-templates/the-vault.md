# Frame — The Vault

> Derived from the Vault archetype (a life-makeover coach). The teaching spine is a set of
> **numbered Secrets unlocked one at a time**, tied together by ONE running metaphor. Every frame
> runs the identical 4-phase skeleton; The Vault only changes the teaching devices, refrains, and
> the close. The non-negotiables (slide ranges, ≥100 floor, ≤2 case studies, 3–7 steps, suggested
> image per slide, N.E.E.I.T./4-Quadrant, Phase-3-no-pitch, central + section hooks) are enforced by
> the provers, not by this template.

- **signature_frame id:** `vault`
- **Teaching unit:** a **Secret** (3–7 total, one per teaching step)
- **Per-Secret shape:** a famous **quote** → the teaching → a numbered **affirmation**
- **Signature devices:** ONE running metaphor motif carried deck-wide (a single controlling image); a **personal-manifesto triad** (a "Unique Passion / Purpose / Power Proposition" statement)
- **Central hook:** the one repeated chorus line; **section hooks:** one per phase

## Phase 1 — Avatar Section (slides 1–11)
- N.E.E.I.T. + 4 Quadrants. Open in the audience's current state: "losing my identity," "just existing," "the little voice," "I know I deserve more."
- Introduce the running metaphor early (it will recur every phase). Section-hook slide for Phase 1.
- Tone ladder: empathetic → relatable → hopeful → motivating.

## Phase 2 — Signature Story (slides 12–24)
- N.E.E.I.T. + 4 Quadrants. The big WHY, family, the schooling chapter, the career climb (e.g. a role → leadership → entrepreneurship arc), the mission.
- Vulnerability + pain-into-purpose; keep the metaphor threaded. End on a gripping "why". Section-hook slide for Phase 2.

## Phase 3 — Transformational Teaching (slides 25–60) — NO PITCHING
- The teaching is **the Secrets**: 3–7 numbered Secrets, each a teaching step.
- Each Secret slide-group: a famous **quote** → Secret title → teaching (mix questions, stories, definitions, formulas, media moments) → a numbered **affirmation**.
- Keep the running metaphor alive in each Secret (e.g. a callback line). Tag **≤2 CASE_STUDY** slides (band 1–2).
- Land the **personal-manifesto triad** (Unique Passion / Purpose / Power Proposition) as the through-line of the section.
- Final teaching step = the bridge: value summary + proof + "what's next" — **no price, no product name, no enroll CTA**.

## Phase 4 — Purpose Pitch (slides 61–100)
- N.E.E.I.T. + 4 Quadrants. Purpose pitch: introduce the offer (exact q7 name), the empowered-self / best-self reframe, benefits, proof, honest scarcity, guarantee, conviction CTAs.
- **Signature close:** a blessing — **"My Prayer for YOU"** — the emotional send-off.

## Refrain & tone policy
- The metaphor is the refrain — one controlling image, reused, never swapped mid-deck.
- Every phase carries its labeled name + purpose slide (Directive 10) before its content.

## Visual identity & suggested-image seed craft (Directive 4)
Every slide's `suggested_image` is the AUTHORING SEED the Prompt Author expands into the
9,000–18,000-char rich prompt — it never replaces that floor, but a strong seed is what makes the
expansion land. Write every seed with the six anchors: **[archetype A1–A5] + subject & emotion +
setting/world + light + palette anchor (from the locked STYLE BLOCK) + composition zone for the
baked copy**. Never put slide copy in the seed (copy is baked at prompt time); describe the PICTURE.

**The Vault's visual world — one metaphor, escalating light:**
- **Controlling visual motif:** the client's ONE running metaphor is also the deck's ONE visual
  through-line. Translate it into a concrete recurring visual element (object, environment, or
  gesture from the metaphor's world) and name that element in seeds across ALL four phases —
  planted in Phase 1, present in every Secret, paid off in the blessing close. Never introduce a
  second metaphor world.
- **Unlock progression:** each Secret "opens" — stage the light to escalate across the Secrets
  (shadowed → cracked-open light → fully lit) so the section visually unlocks as it teaches.
  State each Secret's light stage in its seeds.
- **Quote slides:** every Secret opens on a famous quote — render these as designed-typography
  plates (archetype A4) with the metaphor element as a quiet background texture, consistent
  across all Secrets so the quotes read as a series.
- **Avatar slides (1–11):** "losing my identity" / "just existing" beats in dim, desaturated,
  confined frames; "I know I deserve more" turns the light on — first brand-accent moment.
- **Story slides (12–24):** the climb told in scenes (family, schooling, the career arc); keep the
  metaphor element present in at least one frame per quadrant; the mission beat is wide and lit.
- **Manifesto triad + blessing close:** the triad as three matched designed plates; "My Prayer for
  YOU" is the deck's warmest frame — soft golden light, open composition, benediction energy.
- **Series discipline:** vary archetypes between adjacent slides (anti-template SOP-04) while
  holding palette + metaphor constant; portraits/likeness and the logo ride image-to-image
  references per SOP-IMG-01 — the seed notes WHERE they sit, never re-describes the mark.
