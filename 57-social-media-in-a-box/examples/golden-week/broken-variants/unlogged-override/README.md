# broken-variant: unlogged-override

A caption band was OVERRIDDEN during proving (`working/creative/applied.json`) with NO matching
logged entry in `working/creative/overrides.json`. `build_manifest.py --run-dir run` must refuse
the certificate with **AF-SM-OVERRIDE-UNLOGGED** (exit 2). Deviation is free; a SILENT deviation
is the only forbidden one (creative layer §4.3 step 5).
